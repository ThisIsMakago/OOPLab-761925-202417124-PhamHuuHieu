1. Choosing "Cancle" option makes the program print out "No".
The line:
	option == JOptionPane.YES_OPTION?"Yes":"No"
cause the program to choose the later option, which is "No" when the chosen option is not "Yes".

2. To customize the options, we need to use a more advanced showOptionDialog:

Object[] options = {"I do", "I don't"}; OR Object[] options = {"Yes", "No"};

JOptionPane.showOptionDialog(null,
    [message], 
    [title],
    JOptionPane.YES_NO_OPTION, // Option type
    JOptionPane.QUESTION_MESSAGE, // Message type
    null,
    options, // The button labels
    options[0]); // Default button focus
